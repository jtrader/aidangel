# 水母及海洋生物螫傷

對於熱帶水母（邦德堡以北），將醋倒在螫傷處30秒，然後致電000。對於藍瓶水母及其他溫帶水母的螫傷，清除觸手並浸泡在熱水中。

## 熱帶水母（箱形水母、伊魯康吉水母）

在邦德堡以北的任何地方，應將任何水母螫傷視為可能危及生命。

1. **致電000。**
2. 將**醋**大量倒在螫傷處至少30秒。
3. 如果沒有醋，請用手摘掉觸手（戴手套的手指不會被螫傷；裸露的指尖通常短時間內沒問題）。
4. 準備好開始 [CPR](/kb/cpr) — 箱形水母螫傷可導致心臟驟停。

## 藍瓶水母及其他非熱帶水母

1. 用海水（不是淡水）沖洗患處以清除觸手。
2. **不要**使用醋。
3. 將患處浸泡在**熱水**中20分鐘（溫度不應超過施救者可承受的範圍）。熱水淋浴也有效。
4. 如果沒有熱水，可使用冷敷袋止痛。

## 魟魚、石頭魚、芋螺

將肢體浸泡在**熱水**中長達90分鐘。致電**000** — 這些螫傷可能是致命的。

## 藍環章魚

施行壓力固定法（與處理[蛇咬傷](/kb/snake-bite)相同）並致電**000**。準備好進行CPR — 毒液會使呼吸肌肉麻痺。
