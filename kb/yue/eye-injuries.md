# 眼睛受傷

如果眼睛裡有異物，請用清水或生理鹽水沖洗。切勿移除嵌入的異物 — 遮蓋雙眼並致電000。化學品濺入眼睛需要用流動水沖洗20分鐘並召喚救護車。

## 異物 (沙粒、眼睫毛)

1. 不要揉眼睛。
2. 傾斜頭部，使受影響的眼睛朝下。
3. 用清水、生理鹽水或洗眼液沖洗。
4. 如果異物未能沖出，用潔淨的敷料覆蓋眼睛並尋求醫療協助。

## 嵌入物

1. **切勿**嘗試移除異物。
2. 用紙杯或敷料輕輕覆蓋眼睛 — 不要施壓。
3. **另一隻眼睛也要**蓋上，以防止雙眼移動。
4. **致電000。**

## 化學品濺入

1. **立即**用涼的流動水沖洗眼睛至少**20分鐘**。傾斜頭部，使化學品遠離未受影響的眼睛。
2. 撐開眼瞼。
3. **致電000。**
4. 如果您知道是甚麼化學品，請將容器帶到醫院。

## 眼睛被擊中

敷上冰袋以減輕腫脹。如果視力有任何變化、嚴重疼痛、眼內出血或眼睛看起來異常，請尋求醫療協助。
