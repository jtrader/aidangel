#  choking — 成年人、兒童及嬰兒

如果一個人唔能夠咳嗽、說話或者呼吸，請喺肩胛骨之間大力拍打背部5下，然後按壓胸部5下，並重複交替，直到氣道暢通或者佢哋失去知覺為止。

## 識別 choking

- **輕度（有效咳嗽）：** 患者仍然可以咳嗽、說話或呼吸。鼓勵佢哋繼續咳嗽。
- **嚴重（無效咳嗽）：** 無聲咳嗽、呼吸困難或不能呼吸、用手揸住喉嚨、嘴唇發藍。請立即行動。

## 成年人或兒童（1歲以上）

1. **致電 000** （或請其他人幫忙致電）。
2. 讓患者身體向前傾，用手掌根部喺肩胛骨之間用力拍打背部最多 **5下**。每次拍打後檢查口腔。
3. 如果異物仍未清除，請進行最多 **5下胸部按壓** — 手部姿勢與 CPR 相同，但要動作較慢同更用力。
4. **重複交替** 5下拍背同 5下胸部按壓。
5. 如果患者失去知覺，請小心將佢放喺地面並開始進行 [CPR](/kb/cpr)。

## 嬰兒（1歲以下）

1. **致電 000**。
2. 將嬰兒面部朝下放喺你嘅前臂上，頭部低過胸部，並支撐住頭部同下巴。
3. 喺肩胛骨之間用力拍打背部最多 **5下**。
4. 將嬰兒翻轉過嚟面部朝上，用兩隻手指喺下胸骨處進行最多 **5下胸部按壓**。
5. 重複交替，直到氣道暢通或者嬰兒失去反應 — 然後開始嬰兒 CPR。

## 請勿

- 請勿盲目將手指伸入患者口中 — 你可能會將異物推得更深入。
- 請勿進行腹部衝擊（Heimlich） — 澳洲復甦委員會指引採用背部拍打同胸部按壓。
