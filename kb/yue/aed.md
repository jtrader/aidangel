# AED — Automated External Defibrillator (自動體外心臟去顫器)

AED 可傳遞電擊以使心臟驟停的心臟恢復跳動。現代 AED 專為未經訓練的旁觀者設計 — 打開它，按指示貼上電極片，然後按照語音提示操作。

## AED 的作用

自動體外心臟去顫器 (AED) 會分析心臟的節律，如果檢測到可電擊的節律（心室顫動或無脈性心室頻脈），它會傳遞一次受控電擊，以嘗試恢復正常心跳。

## 如何使用 AED

1. **打開電源。**打開蓋子或按下電源按鈕。
2. **按電極片包裝上的指示**將電極片貼在患者裸露的胸部：
   - **成人：**一塊電極片貼在右鎖骨下方，另一塊貼在左側腋下。
   - **8 歲以下或體重不足 25 公斤的兒童：**如果可用，請使用兒童專用電極片 — 一塊貼在胸部中央，另一塊貼在背部中間。
3. 當 AED 分析心臟節律時，**清場**。
4. 如果收到指示，**實施電擊** — 確保沒有人接觸患者，然後按下閃爍按鈕。
5. 電擊後或如果未建議電擊，**立即恢復 CPR**。AED 將每 2 分鐘重新分析一次。

## 特殊情況

- **皮膚潮濕或下雨：**快速擦乾胸部並繼續。避開水窪。
- **胸部毛髮濃密：**電極片必須接觸皮膚 — 如有需要，請剃毛或快速撕下電極片（大多數套件都包含剃刀）。
- **植入式心臟起搏器：**將電極片放置在距離任何可見腫塊至少 8 厘米的位置。
- **藥物貼片：**移除貼片並擦拭該區域，然後再貼上電極片。

## 尋找 AED

使用 [GoodSAM AED Finder](https://www.goodsamapp.org/locatorMap) 或致電 **000** — 接線員可以引導您找到最近的已註冊 AED。
