# Jellyfish an Marine Stings

Long f’ya olgeta tropical jellyfish (long wup-an-da long Bundaberg), yu mas floodim-awe sting b’fo vinega b’fo 30 sekon, den yu mas callim 000. Long f’ya bluebottle an narra kain sting, yu mas tekem-awe ol tenga an putum-daun insaid hot wata.

## Tropical jellyfish (mas-mas kain jellyfish, Irukandji)

Eniwei wup-an-da long Bundaberg, yu mas treatim eni jellyfish sting olsem hem-kain waan wea kapin killim yu.

1. Yu mas **Callim 000.**
2. Yu mas Pourim **vinega** plentii plentii langa sting b’fo bies 30 sekon.
3. Sapos nobodi gat vinega, yu mas pickim-awe ol tenga (dei gulum-gulum-yu no sapos yu waraim gala, bare finga hem-kain kaina rait b’fo likka bita taim).
4. Yu mas redii b’fo startim [CPR](/kb/cpr) — box jellyfish sting kapin meikim yu hat stop.

## Bluebottle an narra kain jellyfish wea no long tropik

1. Yu mas washim-awe ples b’fo si-wata (no pres wata) b’fo tekem-awe ol tenga.
2. Yu mas **no** usum vinega.
3. Yu mas putum-daun ples insaid **hot wata** (no hotta dan how much di rescuer kapin tolarate-im) b’fo 20 minit. Hot shower hem-kain waka.
4. Sapos nobodi gat hot wata, yu mas usum cold pack b’fo pain.

## Stingray, stonefish, cone shell

Yu mas putum-daun an leg insaid **hot wata** b’fo bies 90 minit. Yu mas Callim **000** — deis kain sting kapin killim yu.

## Blue-ringed octopus

Yu mas applyim pressure immobilisation (olsem b’fo [snake bite](/kb/snake-bite)) an yu mas callim **000**. Yu mas redii b’fo CPR — di venam kapin meikim di braitin masel go paralas.
