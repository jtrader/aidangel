# Sikwan blut suga tabun

Sipos mainwan bin diabitiz bin kofius, bin swat o bin sik, yus supos im blut suga bin daun (hypo) en giwi im suga pud o dring if im kan swela. Tel 000 if im nomo beta, lus main en wokbaot, o luk olsem drai nogud.

## Hypoglycaemia (daun blut suga)

**Saen:** वीकनेस, स्वेटिंग, ट्रेम्ब्लिंग, कंगर, कफ्यूजन, हेडेक, स्लर्ड स्पीच, इरिटेबल बिहेविअर.

**इफ कॉन्शस एंड एबल टू स्वॅलो:**
1. गिउइ **शुगर** — सफ़्ट ड्रिंक (नो डाइट), जूस, जेली बीन्स, ग्लूकोस टेबलेट्स.
2. रिपेट एवरी **15 मिनट्स** अंटी दे रिकॉवर.
3. फॉलो वित अ लोंगर-एक्टिंग कार्बोहायड्रेट (सेंडविच, बिस्कुट).
4. इफ नो इंप्रुवमेंट, **कॉल 000**.

**इफ अनकॉन्शस:** कॉल 000, फॉलो [DRSABCD](/kb/drsabcd), प्लेस इन [रिकवरी पोजिशन](/kb/recovery-position). दू नॉट गिउइ ऐनीथिंग बाई माउथ.

## Hyperglycaemia (haei blut suga)

**Saen:** bin torsti tumas, urin fo lataim, fil sikwan, breth smell olsem prut, bladi — diswan kamap fo naoa awa o dei.

- Follow [DRSABCD](/kb/drsabcd).
- Hap im wit im aan insulin if im ask.
- **Tel 000** if im nomo bata o im bin bladi.

## Wen u bin dout, giwim suga

If yu nomo shur if diswan hypo o hyper, giwim suga. Diswan gon hap hypo en diswan nomo gon mekim hyper wos tumas fo shawt taim.
