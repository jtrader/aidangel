# Spider Bites

Funnel-web an mouse spider bites, yu garra treat im usin pressure immobilisation, same lik wun snake bite. Redback bites, yu garra treat im wit wun ice pack — no put bandage. Go luk wun doctor if eni siknes deya, yu wari.

## Funnel-web o mouse spider

Treat im lik wun big wan — put **pressure immobilisation** (luk [snake bite](/kb/snake-bite)) an ring **000**.

## Redback spider

1. Put wun kol pak (rabinap wit kloth) fo mekim di pen go daun.
2. **No** put wun pressure bandage — em mekim di pen moes yet.
3. Go luk wun doctor. Antivenom deya if yu nid im.

## White-tail, huntsman, ada Australian spiders

1. Wasim di bite wit sop an wota.
2. Put wun ice pack.
3. Go luk wun doctor if di siknes moes bad, di bite bin gat infekshen, o di bite deya on di fes o niar di ain.
