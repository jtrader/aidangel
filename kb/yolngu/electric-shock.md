# Djälk Yolŋuŋuwuy manaŋuny, barrarrawuy manaŋuny

Yaka warray galkinyarawa yolŋuwalany milmaŋuruŋuwuynha manaŋuŋuru. Manaŋuwuy dhawar'yunarawuy, goŋmirriyamany DRSABCD, wäthun 000, gapu-dhawar'yun gawurma'nhany dhuwalany. Yolŋu yaka djäl manaŋuwuynha yukurra warray yakana ganarrthaka yukurra djäl manaŋuwuy gapu-dhawar'yunarawuynha. Manaŋuwuy gapu-dhawar'yunnha nhina rirrakay-dhunarawuynha mala. Manaŋuwuy gapu-dhawar'yum djäl nhini yukurra rirrakaynydjaŋuruŋuy manaŋuŋuru.

## Buŋgulnydja mala dhuwalany

- **Yaka warray galkinyarawa yolŋuwalany milmaŋuruŋuwuy manaŋuŋuru.** Manaŋuny dhuwalany yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
- Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
- Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.

## Gaŋguriŋuŋuy dhuwalany

1. Manaŋuwuy dhawar'yunarawuy, manaŋuny DRSABCD. Manaŋuwuy dhawar'yunarawuy, manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
2. **Wäthun 000.**
3. Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny CPR manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny AED.
4. Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny burn manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
5. Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny shock manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
6. Manaŋuwuy dhawar'yunarawuy manaŋuny yukurra warray yolŋuwalany manaŋuwuy dhawar'yunarawuy. Manaŋuwuy dhawar'yunarawuy, manaŋuny manaŋuwuy dhawar'yunarawuy manaŋuny yakana manaŋuwuy dhawar'yunarawuy manaŋuny.
