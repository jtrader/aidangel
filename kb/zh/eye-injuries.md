# 眼睛受伤

眼睛里进了东西，用清水或生理盐水冲洗。切勿取出嵌入物 —— 遮盖双眼并拨打000。化学品溅入眼睛需要用流动水冲洗20分钟并叫救护车。

## 异物（沙粒、睫毛）

1. 不要揉眼睛。
2. 头部倾斜，使受影响的眼睛向下。
3. 用清水、生理盐水或洗眼液冲洗。
4. 如果未能冲出，用干净的垫子盖住眼睛并寻求医疗帮助。

## 嵌入物

1. **不要**试图取出它。
2. 用纸杯或垫子轻轻盖住眼睛 —— 不要施加压力。
3. **另一只眼睛也**要遮盖，以防止两只眼睛移动。
4. **拨打000。**

## 化学品溅入

1. **立即**用凉的流动水冲洗眼睛至少**20分钟**。头部倾斜，使化学品远离未受影响的眼睛。
2. 保持眼睑张开。
3. **拨打000。**
4. 如果知道是什么化学品，请将容器带到医院。

## 眼睛受到撞击

敷冷敷包以减轻肿胀。如果视力有任何变化、剧烈疼痛、眼睛内部出血或眼睛看起来异常，请寻求医疗帮助。
