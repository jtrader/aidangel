# Kuruwalku

Murtilya manta mulapa ngurtungurtu-nguru yankula ngarntarringkulaya kapi pipirringara pikila ilaringkula ngarrinytjaku, palyara kapi ngarntarranganta kapi pilpirringara kapi ngarrinytja ilaringkula. Palya kuruwalku—ngampalala, kapi ngarrinytja kulila, pikilaya ilaringkula—palya yankula ambulance-ngarntalu yaanparra.

## Kanyininpa

- Kapi tjukuritja, ngalypuru walara
- Kapi ngarrinytja pulyura walara, uti kapi ngarrinytja kulila
- Kuru palya, murtinya ilaringkula, malurru
- Kuru kapi walara, kurra ngalypuru walara
- Pikilaya ilaringkula, ngarrinytja ilaringkula (palya)

## Wiya yarratjanu

1. Murtilya manta mulapa ngurtungurtu-nguru yankula ngarntarringkula.
2. Kapi pipirringara pikilaya ngarrinytja ilaringkula — kapi pipirringara mulapa tjukurrpa ilaringkula pikilaya ngarrinytja ilaringkula palyaku.
3. Caffeine, alcohol kapi kapi tjukurrpa ngarrinytja pilpirringara pikila ilaringkula ngarrinytjaku.
4. Kuruwalku tjukurrpa ilaringkula ngarntarra, palyaku kapi pipirringara pikilaya ngarrinytja ilaringkula (e.g. Hydralyte).
5. **000-ngarntalu yaanparra** nintilu murtilya ngampalala, kapi ngarrinytja kulila, pikilaya ilaringkula, uti ngawuru.
