# DRSABCD Action Plan

DRSABCD action plan-ungka Australian Resuscitation Council-pa walytjapiti tjukarurrunya mantjira ananyi palulakurra tjinguru tjana tjiwangka pakanma: Danger, Response, Send for help, Airway, Breathing, CPR, Defibrillation.

## DRSABCD-pa palula kanyini palatja

DRSABCD (talingka pitingka pulka DRSABCD) pulka walytjapiti tjukarurrunya mantjira ananyi palulakurra tjana tjiwangka pakanma Australians-pa pulka walytjapiti tjukarurrunya. Palula pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjinguru tjana tjiwangka pakanma. Kanya walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma First Aid-pa pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.

1. **D — Danger.** Tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma ananyi pulka walytjapiti tjukarurrunya: pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma. Walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma: pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pitingka pulka walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.
2. **R — Response.** Patila yalatji. Wangka nguru. '*Ngani ngunytjiwa? Wiira nguntjananya. Ngunytjiwa nguru patila walytjapiti tjukarurrunya.*' COWS-pa pulka walytjapiti tjukarurrunya: Can you hear me? Open your eyes. What's your name? Squeeze my hand.
3. **S — Send for help.** Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma **000** for an ambulance. Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma: pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.
4. **A — Airway.** Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma. Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma. Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.
5. **B — Breathing.** Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma, pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma tjinguru **10 seconds**. Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.
6. **C — CPR.** Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma. **30 chest compressions to 2 rescue breaths** at a rate of 100–120 compressions per minute.
7. **D — Defibrillation.** AED-pa pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.

## Important reminders

- Pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma. [recovery position](/kb/recovery-position) nguru pulu walytjapiti tjukarurrunya mantjira ananyi palula tjana tjiwangka pakanma.
- Do not interrupt chest compressions for more than 10 seconds at a time.
