# Ninti Pukutjaratjara

Tjinguru pukultu katja-katjara, yalatji kutjupalu yankulawiyangku. Pikatjarra, mungartu kapi kutjupalu palyanma, kapi kutjupalangka palyanma. Tjina pika tjanangka palyanma.

## Pikatjarra (pika-pika, pika-pika kutjupa)

1. Wiya pika wiya.
2. Pika wiya wiya. Pika wiya wiya. Pika wiya wiya.
3. Pika kapi kutjupalu palyanma. Pika kapi kutjupalu palyanma.
4. Wiya witula, pika kapi kutjupalu palyanma, kapi kutjupalu palyanma.

## Pikatjarra tjanangka

1. Wiya witula tjanangka pikatjarra.
2. Pika wiya tjanangka pikatjarra, kapi kutjupalu palyanma. Pika wiya wiya.
3. **Kapi kutjupalu palyanma** tjanangka pikatjarra.
4. **000-pa kanyini.**

## Pikatjarra kapi kutjupalu

1. **Kapi kutjupalu palyanma** tjanangka pikatjarra, kapi kutjupalu palyanma, **20 minutes-angka** palyanma. Pika wiya tjanangka pikatjarra, kapi kutjupalu palyanma.
2. Pika wiya tjanangka pikatjarra.
3. **000-pa kanyini.**
4. Wiya tjanangka pikatjarra, kapi kutjupalu palyanma, kapi kutjupalu palyanma.

## Pikatjarra tjanangka

Pika wiya tjanangka pikatjarra. Pika wiya tjanangka pikatjarra, kapi kutjupalu palyanma, kapi kutjupalu palyanma, kapi kutjupalu palyanma.
