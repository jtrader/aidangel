# Tjukuṟuru pakani

Tjukuṟuru pakaningku tjuntalpa kapi tjana ananyi pika ngarringi kutju 35°C wiyangku. Wiṟumpaku katjaṟa ngura walytjaku, kapi tiwiṟu yankupai wata tjankaṟa, munu wiṟuwiṟu tjankaṟa pika ngarringi, munu 000-ku waakaṟa kulila palya wiṟunguru.

## Kuka tjunu

- Tjukuṟuru (palya kutju ngarringi kulila wiyangku)
- Kapi tjukuritja, ngamu kapi tjankaṟa
- Walytja wata tjankaṟa
- Ngalyipiṟi walytja kapi tjankaṟa, kapi tjankaṟa, walytja wata tjankaṟa
- Ngamu munu walytja kapi tjankaṟa

## Kuka tjunu

1. **000-ku waakaṟa kulila** palya wiṟunguru.
2. Wiṟumpaku katjaṟa ngura walytjaku, kapi tiwiṟu yankupai wata tjankaṟa.
3. Walytja kapi tjankaṟa — kutju tjunkalani tjukaṟuru kapi tjankaṟa kulila palya wiṟunguru.
4. Kapi tiwiṟu yankupai wata tjankaṟa; kutju tjunkalani tjukaṟuru kapi tjankaṟa palya wiṟunguru.
5. Tjukaṟuru kapi tjankaṟa palya wiṟunguru.
6. Wiṟuwiṟu **tjankaṟa** tjukaṟuru kapi tjankaṟa palya wiṟunguru, munu kapi tjankaṟa palya wiṟunguru.
7. Tjukaṟuru **kapi tjankaṟa** kutju tjunkalani tjukaṟuru kapi tjankaṟa palya wiṟunguru. **Kapi tjankaṟa wata tjankaṟa.**
8. Kapi tjankaṟa wata tjankaṟa kutju tjunkalani tjukaṟuru kapi tjankaṟa palya wiṟunguru.
9. CPR kutju tjunkalani tjukaṟuru kapi tjankaṟa palya wiṟunguru — kutju tjunkalani tjukaṟuru kapi tjankaṟa palya wiṟunguru.
