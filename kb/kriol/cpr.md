# CPR — Majabana Stat Mada

CPR emi ola langa eniwan hu kani breth normalijat o hu nomo riyak. Yu kani yusa 30 kist kompresan tu 2 brej, langa 100-120 kompresan pur minit. Mani di dipnes an di wai yu duuwum langa bigman, lilwan an beibi.

## Wen yustat CPR

Stat CPR langa samwan hu **nomo riyak an nomo breth normalijat.** Nomijat wet - ebri minit witout CPR emi kani lowurim di chansus fo laif baiaut 10%.

## Bigman an lilwan (oba wan yiya)

1. Lec di pesin bin le daun langa strong pleis an go naidan langa di kist blonga dem. 
2. Putum di hil blonga wan hand blongyu langa di **lomeda haaf blonga di kist** (santur blonga di kist). Putum di oja hand blongyu oba an tinggum tingga.
3. Kip yuself straet an di sholda blongyu oba yuhan.
4. Push daunn **wan tud blonga di kist dipnes** (raun 5 cm langa bigman).
5. Push langa reit blonga **100-120 pur minit** - raun 2 pur sekan.
6. Gib **30 push**, den **2 riprizentatif ripizit**.
7. Gohed 30:2 majabana til di pesin bin geda, ambulens bin kam, AED bin tok yu stop, o yu kani gohed.

## Beibi (anda wan yiya)

1. Putum di beibi langa strong pleis.
2. Yusa **tu fingga** langa di lomeda haaf blonga di kist.
3. Push daunn wan tud blonga di kist dipnes langa 100-120 pur minit.
4. Fo breth, gabarum **mauf an nos** blonga di beibi wit yamaf an gib lil buf - jast inaf fo si di kist go up.
5. Gohed 30:2.

## CPR - jast pushin

If yu kani gib breth o yu kani, duuwum **gohed kist pushin** langa 100-120 pur minit. Kompresan-jast CPR emi beta tumaj dan nomo CPR.

## Wen yu stop

Stop CPR only if:
- Di pesin bin stat breth normalijat.
- Wan paralegal o dokta bin tek oba.
- Wan AED bin tok yu fo stan klin (gohed raitwei if im bin tok yu).
- Yu bin taia tumaj an nomo eniwan bin kani tek oba.

## Si olso

CPR emi step **C** blonga [DRSABCD](/kb/drsabcd). Putum wan [AED](/kb/aed) raitwei wen im bin kam - nomijat stop CPR fo wet fo im.
