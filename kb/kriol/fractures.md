# Brukun Bun (Bun Brok)

Holim di pat we bin bin brok stil. Stopim eni blad we kambek. Holim im stil langa wan splint o sling we guda bin padim. Sekim blad de go ron gudu o nogo ebri 15 minit. Go luk medikel help. Gobek 000 if di bun brok owpen, oleta bin brok o yu lukim di pesin showim wan samting olsem shok.

## Sain

- Pen, espeshali wen yu muwum
- Tenades owa di bun
- Swelin en bruzin
- Brok o bin bin bin go long narrawei
- Kan yuzum o putum wait langa im
- Mekin nois o yu bin pilim wan samting de grindim
- Owpen waund wit bun yu bin lukum (owpen bun brok)

## Olabat we yu garra du

1. Folowim [DRSABCD](/kb/drsabcd).
2. Holim di pesin stil en mekim im pilim seif.
3. Stopim eni blad we de kamaut wit stret presha (raun wan owpen bun, norra langa im).
4. **Holim im stil** di pat we bin bin brok langa di samwei we yu bin lukim, yuzum ol padded splints o yu bin tainim im langa di bodi / narra pat.
5. Yuzum wan sling fo apalang brok, if yu bin bin kan.
6. Sekim di blad de ron gudu o nogo afta di injuri (hotnes, kala, pouls) ebri **15 minit**. Luwusim ol bandages if yu bin bin pilim namnes o kolnes.
7. Tritim fo [shok](/kb/shock).
8. Go luk medikel help. Gobek **000** fo ol bikpala bun (femur, pelvis), owpen brok, plenti brok o yu bin lukim di pesin showim wan samting olsem shok.

## Norra du

- Norra traim fo pushum di bun bak long im pleis.
- Norra gibit fut o wata — yu garra nidim sirjeri.
- Norra muwum di pesin longtaim we yu norra nidim.
