# Jampella Wata

Tekem detwan autsaid di wata, sista if im seif fo yu fo dum. Statim CPR hapa taim if detwan nau sabi neitin an nau bretin normal - kolem 000 fo ebriwan ho jampella wata, sista if im luk olraet bakgen.

## Wanem fo dum

1. **Kolem 000** fo ebriwan ho jampella wata.
2. Tekem autsaid di wata — **nau yusap yuwan laif**.
3. Folom [DRSABCD](/kb/drsabcd).
4. Putum deyan on im bek. If im nau bretin normal, statim [CPR](/kb/cpr) hapa taim — gib **5 fo statim bret fo helpum** bifo yu stat pressim im ches.
5. Mekem yusap yo redi fo ol detwan ho bomitim — trenim deyan on im said an klirim im maut.
6. Kipgon CPR sista detwan sabi bakgen o deyan from hospital kam.
7. **Na** traim fo tekem wata autsaid di belijing fo pressim deyan on di belijing.

## Jampella wata bakgensaid

Ebribodi ho neya jampella wata mas go fo hospital fo deyan lukluk deyan, sista if deyan luk olraet. Wata insaid di langas ken mek bigbig trabul lata an im ken kilim deyan.

## Kulwan Bodijing

Ol detwan ho deyan seybim from kul wata oltaim deyan kulwan bodijing — lukim [kulwan bodijing](/kb/hypothermia). Kipgon CPR fo longata taim den yu oltaim dum; deyan don mekem deyan sabi bakgen afta deyan bin longtiam insaid kul wata.
