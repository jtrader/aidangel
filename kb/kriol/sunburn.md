# Sanban

Muwuwei longta san, kipum di area wit kold woda fo 20 minit, pudum afta-san jel, dringam brabli woda, en neba bosim blista. Luk medikel elp fo nogud sanban o blista pasin on pikinini.

## Wanem fo dum

1. Muwuwei longta san.
2. Kipum di skin we gad sanban anda **kold woda we ranan fo 20 minit**.
3. Pudum afta-san jel, alo vera olot laika wan lotion we meigim skin sof.
4. Dringam brabli woda fo meigim yu fil beta gens.
5. Kobaram bifo yu go owt olgatawan gens.
6. **No bosim blista.**

## Luk medikel elp if

- Di sanban kaba brabli bigwan area
- Yu gad brabli plenti blista
- Diwan we sanban na beibi, liklwan pikinini o olgaman
- Dei gad signa fo [hit siknes](/kb/heat-illness) — hai tempraja, womit, dizines, konfyuwsan
